소개

안녕하세요. 백엔드 개발자 홍길동입니다. 계명대학교 컴퓨터공학과를 졸업하고, 현재 DCM에서 백엔드 개발자로 활동하고 있습니다. 대규모 트래픽 처리와 분산 시스템 설계에 관심이 많으며, 최신 기술 트렌드를 꾸준히 학습하고 있습니다. 사용자 중심의 서비스 개발을 지향하며, 팀원들과의 원활한 협업을 통해 최상의 결과물을 만들어내는 것을 목표로 하고 있습니다.

주요 프로젝트

WEGO (스마트오더 앱)

wego 스마트오더 : 프로젝트 소개

기간: 2024.02 - 2024.06 (5개월)

WEGO는 사용자 위치 기반의 스마트오더 서비스로, 편리한 주문과 결제 경험을 제공합니다. 이 프로젝트에서 백엔드 개발을 담당하여 안정적이고 확장 가능한 서버 시스템을 구축했습니다.

주요 기능 및 성과:

  • GPS 데이터를 활용한 실시간 매장 검색 및 추천 시스템 구현
  • 여러 PG사 연동을 통한 다양한 결제 옵션 제공 (신용카드, 간편결제 등)
  • Redis를 활용한 주문 데이터 캐싱으로 응답 시간 40% 개선
  • RabbitMQ를 이용한 비동기 주문 처리로 시스템 안정성 향상
  • Docker와 Nginx를 활용한 마이크로서비스 아키텍처 구현으로 서비스 확장성 확보
  • Firebase Cloud Messaging을 활용한 실시간 푸시 알림 시스템 구축

사용 기술:

Java Spring Boot JPA Redis RabbitMQ MySQL Docker Nginx Firebase

스마트 팩토리 관리 시스템

기간: 2024.06 - 2022.07 (1개월)

제조 공정의 효율성을 극대화하기 위한 스마트 팩토리 관리 시스템을 개발했습니다. 이 시스템은 실시간 데이터 수집 및 분석을 통해 생산성 향상과 비용 절감을 실현했습니다.

주요 기능:

  • 실시간 원자재 수량 파악 시스템
  • 완성품 수량 실시간 모니터링
  • 완성품 추적 시스템
  • 생산 라인별 실시간 모니터링
  • 대시보드를 통한 재고 현황 실시간 시각화

성과:

  • 원자재 관리 효율성 30% 향상
  • 작업자 오류 15% 감소

사용 기술:

Java Spring Boot JPA MySQL Docker
